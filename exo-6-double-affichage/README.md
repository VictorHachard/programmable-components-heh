# Note
Exercice du camarade correspondant au PSF 6 (5 bits sur deux afficheurs 7 segments).
Differences avec le PSF : l'entree est un compteur 24 bits (non un bus 5 bits), et il n'y a qu'un seul afficheur 7 segments (non deux).
Le decodeur SSegments (PSF 5) est bien instancie dans ce circuit.
